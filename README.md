# UEVR.Frontend

This fork has been expanded with the following features:

Automatic injection into all Unreal games with background idling

System tray/notify icon with status indicator and context menu

Window can be minimized entirely to system tray with identical behavior to Steam or Epic

Usermode injection with Windows startup fixes for Steam and Epic launchers

Automatic installation of the mod and setup of proper folders

Automatic Nightly Updates

Specific Nightly Version Installer

Automatic profile cleanup

Early injection when run as administrator (may bypass security measures)

AI was used primarily for XAML which is a horrendous language to work in and for writing pinvoke functions
